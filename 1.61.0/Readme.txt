Updated Turkish translation pack.

version=3450

Both po and internal directory txt files are translated. Extract the zip file and upload the lang folder into
/usr/local/directadmin/data/skins/evolution path, overwrite if files are exists.

Don't forget to check directory owner and permissions as parent folder.

If you used any Turkish translation before and feel the skin translation is not updated, then delete login-tr.json and tr.json files from /usr/local/directadmin/data/skins/evolution path. They will be recreated when you refresh DirectAdmin panel page in web browser.


----- Turkish -----

po ve internal klasöründeki metin dosyaları çevrilmiştir. Zıp dosyasını ayıkladıktan sonra 
lang klasörünü /usr/local/directadmin/data/skins/evolution yoluna yükleyin, var olan dosyaların üzerine yazın.

Klasör sahibini ve izinlerini üst klasör ile aynı olacak şekilde ayarlamayı unutmayın.

Daha önce herhangi bir Türkçe çeviri kullandıysanız ve yükleme sonrası tema çevirisinin güncellenmediğini düşünüyorsanız 
/usr/local/directadmin/data/skins/evolution yolundaki login-tr.json ve tr.json dosyalarını silin. Web tarayıcıdaki DirectAdmin panosu sayfasını yeniden yüklediğinizde bu dosyalar yeniden oluşturulur.
